Endpoints

API reference

GET /api/search

Modern search endpoint. Accepts query or q. Add details=true for album and playlist track lists.

curl "http://localhost:5575/api/search?query=slow%20motion"
GET /api/health

Health endpoint for Vercel checks and uptime monitors. Add upstream=true to test the JioMusic upstream.

curl "http://localhost:5575/api/health"
GET /result/

Compatibility route for old clients from the original project. Returns the raw result payload.

curl "http://localhost:5575/result/?query=slow%20motion"
